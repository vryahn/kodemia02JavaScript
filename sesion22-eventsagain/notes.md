# Sesión 22. Eventos y Formularios

## Eventos

#### keydown / mousedown

Ocurre cuando la tecla/botón se presiona y llega abajo

#### keyup / mouseup

Ocurre cuando se presiona la tecla/botón y este llega arriba nuevamente
